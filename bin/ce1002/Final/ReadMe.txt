Java版本: 11.0.2
所用套件:javafx.controls, javafx.fxml, javafx.media
操作方法: 遊戲分成 easy, hard和 impossible 三種難易度，如果再選擇畫面按下鍵盤A，則可以使用沙盒模式。
	這是一款舊時代的貓咪大戰爭，小時候電腦課我都會玩這款遊戲，Age of War中文翻譯為戰爭進化
	這是一個塔防遊戲，玩家的目的是要摧毀敵方的基地，同時也要守護我方的基地
	遊戲時代分為三個，分別是 原始時代 城堡時代 和 工業時代，玩家可透過擊殺敵方士兵來累積經驗值並升級，每次升級可為我方的基地加血
	每個時代有三種不同的兵種，分別是 進戰兵 遠程兵 和特殊兵種，製造士兵需要金錢，金錢可由擊殺敵方的士兵獲得，這些士兵分別是
		原始時代 :
			 club amn -> 手拿棒子的原始人，攻速慢 血量低 免洗兵種
		    slingshot man -> 手拿投執行武器，攻速快 攻擊力低
		       dino rider -> 龍騎士，攻速慢 血量高 攻擊力高
		城堡時代
			sword man -> 手拿沉重的鐵劍，攻速慢
			 arch man -> 手拿十字弩，攻速快
			   knight -> 騎士，攻速慢 血量高 攻擊力高
		工業時代
			knife man -> 手拿軍用小刀，攻速快 殺傷力高
		      AKM soldier -> 手拿AKM，攻速快 殺傷力高
			     tank -> 坦克，攻速慢 高血量 和毀滅性的傷害
	三個時代又有屬於各自的技能，分別是
		原始時代 火山爆發 -> 召喚火山噴發來擊殺敵方士兵
		城堡時代 狂暴號角 -> 吹響號角增加我方士氣，提升我方士兵的攻擊力 攻擊速度和體型
		工業時代 飛彈轟炸 -> 呼叫附近的直升機增援，沿路對敵方進行轟炸
	只要任何一方的基地血量歸零，遊戲結束
